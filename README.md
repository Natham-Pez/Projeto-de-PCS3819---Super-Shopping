# Projeto-de-PCS3819---Super-Shopping
Projeto relativo à disciplina de PCS3819 - Aplicações e Tecnologias em Automação
# Instruções de uso
Para rodar a aplicação, vá para o diretório ./superShopping, abra um terminal e digite "npm run dev", para a aplicação no navegador da sua máquina por meio do localhost do vite. Ou, rode "npm run dev -- --host", para que seja possível acessar a aplicação por meio de um endereço de IP público.

# Observações
Usando "npm run dev -- --host" é possível acessar a aplicação pelo celular *se os dois dispositivos estiverem conectados na mesma rede*
